# Buscando Figuras V4.1

## Para ejecutar el Programa:

- Descargue este repositorio en su equipo.

- Copiar la Carpeta "BuscandoFiguras4" (en la raíz del repositorio) en la raíz del disco C: de su equipo --- Debe quedar la dirección -> "C:\BuscandoFiguras4" --- Si no se ha copiado esta carpeta que contiene el audio NO SE ESCUCHARÁ el RUIDO ROSA durante la exploración.

- Inicie el ejecutable que se encuentra en la carpeta "bin/debug" del repositorio.

## Para editar el Código (debe tener Microsoft Visual Studio instalado en su equipo):

- Descargue este repositorio en su equipo.

- Inicie Microsoft Visual Studio -> En barra de tareas -> FILE -> Project/Solution

- Seleccione el archivo "BuscandoFiguras.csproj" en la raíz del repositorio y se abrirá todo el proyecto automáticamente.
